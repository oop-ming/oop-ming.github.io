Grundlagen der Java-Programmierung

Einführung – Der Übergang von Processing zu Java

Im ersten Semester hast du bereits mit Processing gearbeitet und grundlegende Programmierkonzepte kennengelernt. Processing basiert auf Java, ist jedoch speziell darauf ausgelegt, kreative und grafische Anwendungen einfach umzusetzen. In diesem Kapitel lernst du, wie du von Processing zu „reinem“ Java übergehst und die Grundlagen der Java-Programmierung meisterst.

Java ist eine der weltweit am häufigsten verwendeten Programmiersprachen. Sie wird in Bereichen wie Webentwicklung, Mobile Apps, Unternehmenssoftware und sogar in eingebetteten Systemen eingesetzt. Der Übergang von Processing zu Java ermöglicht dir nicht nur, komplexere Anwendungen zu entwickeln, sondern auch, deine Programmierkenntnisse auf ein professionelles Niveau zu heben.

1. Einrichtung der Java-Entwicklungsumgebung

Um mit Java zu arbeiten, benötigst du eine Entwicklungsumgebung (IDE). Die gängigsten Optionen sind IntelliJ IDEA, Eclipse und NetBeans. Für dieses Skript verwenden wir IntelliJ IDEA.

Schritte zur Einrichtung:

  1. Java Development Kit (JDK) installieren
    Lade das JDK von der offiziellen Oracle-Website oder einer OpenJDK-Quelle herunter und installiere es. Das JDK enthält alles, was du brauchst, um Java-Anwendungen zu schreiben und auszuführen.
  2. IntelliJ IDEA installieren
    Lade IntelliJ IDEA (Community Edition) herunter und installiere es. Diese IDE bietet viele hilfreiche Funktionen wie Syntaxhervorhebung, Autovervollständigung und Debugging-Tools.
  3. Erstes Projekt erstellen
    • Öffne IntelliJ IDEA und erstelle ein neues Projekt.
    • Wähle „Java“ als Projekttyp aus und stelle sicher, dass dein JDK korrekt verknüpft ist.
    • Erstelle eine neue Klasse mit dem Namen Main.

2. Erste Schritte: Dein erstes „Hello World“-Programm

Ein klassisches Beispiel für den Einstieg in eine neue Programmiersprache ist das „Hello World“-Programm. Es zeigt dir die grundlegende Struktur eines Java-Programms.

public class Main {
   public static void main(String[] args) {
      System.out.println("Hello World!");
   }
}

Erklärung:

  • public class Main: Jede Datei in Java enthält mindestens eine Klasse. Hier heißt die Klasse Main.
  • public static void main(String[] args): Dies ist die Hauptmethode, die beim Start des Programms ausgeführt wird.
  • System.out.println("Hello World!");: Gibt den Text „Hello World!“ in der Konsole aus.

Vergleich zu Processing:
In Processing würdest du einfach println("Hello World!"); schreiben. In Java ist die Struktur strenger: Du musst Klassen und Methoden definieren.

3. Grundlegende Syntax und Datentypen

Syntax

Die Syntax von Java ähnelt anderen C-basierten Sprachen (z. B. C++). Hier einige wichtige Punkte:

  • Semikolons (;) beenden Anweisungen.
  • Blöcke werden durch geschweifte Klammern {} definiert.
  • Kommentare:
    • Einzeilig: // Kommentar
    • Mehrzeilig:
      /* 
      Mehrzeiliger Kommentar 
      */

Datentypen

Java unterscheidet zwischen primitiven Datentypen und Referenzdatentypen.

Primitiver Typ Beschreibung Beispiel
int Ganze Zahlen int x = 10;
double Gleitkommazahlen double y = 5.5;
char Einzelne Zeichen char c = 'A';
boolean Wahrheitswerte boolean b = true;

5. Kontrollstrukturen

Kontrollstrukturen steuern den Ablauf deines Programms.

If-Anweisung

int x = 10;
if (x > 5) {
   System.out.println("x ist größer als 5");
} else {
   System.out.println("x ist kleiner oder gleich 5");
}

For-Schleife

for (int i = 0; i < 10; i++) {
   System.out.println(i);
}

While-Schleife

int i = 0;
while (i < 10) {
   System.out.println(i);
   i++;
}

Vergleich zu Processing:

Die Syntax für Kontrollstrukturen bleibt nahezu identisch zwischen Processing und Java.

6. Arrays

Arrays sind Sammlungen von Elementen desselben Typs.

int[] zahlen = {1, 2, 3, 4, 5};
System.out.println(zahlen[0]); // Gibt "1" aus

  2. Erstelle ein Array mit den Zahlen von 1 bis 10 und gib alle Zahlen in umgekehrter Reihenfolge aus.

Zusammenfassung

  • Java ist eine objektorientierte Programmiersprache mit strengerer Syntax als Processing.
  • Ein Programm besteht aus Klassen und Methoden; die Hauptmethode heißt main().
  • Kontrollstrukturen wie if, for und while ähneln denen in Processing.
  • Arrays sind Sammlungen gleichartiger Elemente und müssen in ihrem Typ definiert werden.
  • Das „Hello World“-Programm zeigt die grundlegende Struktur eines Java-Codes.
